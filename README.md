### ⚙️ How to Run


2. Navigate to the project folder:
   ```bash
   cd python-project
   ```

3. Run the project file:

   ```bash
   python filename.py
   ```

Make sure you have Python installed on your machine. You can download it from [here](https://www.python.org/downloads/).
